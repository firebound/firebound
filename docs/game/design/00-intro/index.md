# Introdução

Bem-vindo à documentação de game design **do jogo FIREBOUND**!

Aqui você encontrará as ideias, conceitos e diretrizes que orientam a criação do universo, das mecânicas e da experiência do jogo.

Esta seção serve como ponto de partida para entender a visão geral do jogo, os temas principais, as inspirações culturais e as escolhas de design que tornam a experiência única. O objetivo é facilitar a colaboração entre todos os envolvidos, oferecendo um guia claro e acessível para consulta e referência durante o desenvolvimento.

:::info

Este documento serve apenas para **descrever** conceitos, ideias e diretrizes gerais do jogo.

Nenhuma implementação, especificação técnica ou definição de como apresentar esses conceitos deve estar presente aqui. Todas as decisões de implementação e detalhes específicos devem ser tratados nos documentos relacionados ao conteúdo em si. Para isso, consulte a [Documentação de Conteúdo](../../content/00-intro/index.md).
:::
