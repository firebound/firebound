# Ações

As ações representam tudo o que uma personagem pode fazer durante o jogo, especialmente em combate.

As ações são fundamentais para a dinâmica do jogo, permitindo que os jogadores interajam com o ambiente, realizem ataques, utilizem habilidades e tomem decisões estratégicas.

Possuem:

- [Tipo](00-action_types.md): Classificação das ações em básicas, habilidades básicas e habilidades especiais.
- [Categorias](01-action_categories.md): categorização em ofensivas, defensivas, de movimento, etc.
- [Custo](02-action_cost.md): Consumo de energia necessário para realizar cada ação.
- [Efeitos](03-action_effects.md): Impactos das ações, como dano, cura ou aplicação de efeitos de status.
- [Posicionamento e Alvo](04-action_position_and_target.md): Requisitos de posicionamento no campo de batalha para realizar a ação.
