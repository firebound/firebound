# Introdução

Aqui você encontrará uma visão detalhada e organizada do jogo, com informações sobre o mundo, personagens e outros elementos que compõem toda a experiência do jogador. Esta seção serve como um *catálogo* para documentar tudo o que foi implementado e como cada parte se conecta.

É importante notar que esta seção não descreve as mecânicas de jogo, mas sim fornece detalhes sobre o conteúdo do jogo e como ele é estruturado. Para informações sobre as mecânicas de jogo, [consulte sobre as mecânicas que o jogo possui](../../design/02-mechanics/index.md) na seção de game design.
