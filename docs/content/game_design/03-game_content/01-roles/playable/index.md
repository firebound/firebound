# Papéis Jogáveis

Os papéis jogáveis representam os arquétipos e funções que os personagens podem assumir no jogo. Cada papel define um conjunto único de habilidades, atributos e estratégias, permitindo que os jogadores escolham estilos de jogo que melhor se adaptem às suas preferências. Esses papéis são projetados para trabalhar em sinergia, promovendo a colaboração e a criação de táticas diversificadas durante os combates e desafios do jogo.
