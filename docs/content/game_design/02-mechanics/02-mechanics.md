# Mecânicas Principais

Esta seção apresenta as principais mecânicas do jogo, servindo como guia para o desenvolvimento das regras, sistemas e interações que definem a experiência do jogador.

Aqui você encontrará descrições dos sistemas centrais, como personagens, exploração, batalhas, progressão e inventário. O objetivo é garantir clareza e alinhamento entre todos os envolvidos, facilitando a implementação e evolução das mecânicas ao longo do projeto.

Use este documento como referência para entender como cada parte do jogo funciona e como elas se conectam para criar uma experiência envolvente e divertida.
