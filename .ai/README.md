# .ai

Casa de **convenções** e **skills** do projeto, voltada a agentes de IA (e humanos).

A porta de entrada geral é o [`AGENTS.md`](../AGENTS.md) na raiz. Este diretório
detalha o "como": padrões a seguir e capacidades reutilizáveis.

## Conteúdo

- [`conventions/`](conventions/) — padrões do projeto (monorepo, código, commits, Godot).
- [`skills/`](skills/) — skills específicas do projeto (vazio por ora).

## Relação com AGENTS.md

`AGENTS.md` dá o panorama (o que é o projeto, layout, como rodar). `.ai/` aprofunda
as regras e procedimentos. Quando os dois divergirem, `AGENTS.md` aponta para a
fonte detalhada aqui.
